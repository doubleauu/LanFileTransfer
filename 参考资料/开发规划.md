# 局域网文件传输系统开发规划

## 1. 开发总目标

- [x] 完成一个基于 C/S 架构的局域网文件传输系统
- [x] 使用 `TcpClient` 实现客户端连接
- [x] 使用 `TcpListener` 实现服务端监听
- [x] 使用 EF Core 实现数据库访问
- [x] 使用 SQLite 保存用户、文件和传输记录
- [x] 使用 WinForms 实现图形化客户端
- [x] 支持单文件上传和下载
- [x] 支持文件夹 ZIP 上传和下载
- [x] 支持上传/下载进度显示

## 14. 阶段十三：多客户端支持（已舍弃）

本项目是课程大作业，不继续做多客户端加固。

当前服务端已经使用 `Task.Run` 处理客户端连接，作为基础并发说明即可，不再新增连接管理、并发队列、数据库并发优化等工程化内容。

---

## 15. 阶段十四：异常处理加固（已舍弃）

本项目只保留已有的基础错误提示和日志输出，不单独做异常处理加固阶段。

后续如果发现明显影响演示的问题，只在对应功能附近做小修补，不新增统一异常框架。
